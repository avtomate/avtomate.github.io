Enumerable#map and You

February 4, 2015

The .map method allows one to execute a given block (e.g. { |x| x * 2 } ) for each element of the array. That sounds awfully similar to the .each method, no? True! The main difference being that .map returns a new array whose values are the return values of each block iteration, whereas .each will return the original, unmodified array by default.

Enumerable#map in Practice

yeezy_goat_albums = ["My Beautiful Dark Twisted Fantasy ", "Graduation ", "Yeezus "]
all_caps_weee = yeezy_goat_albums.map { |album| album.upcase }
print all_caps_weee
#MY BEAUTIFUL DARK TWISTED FANTASY GRADUATION YEEZUS

As you can see, this method manipulates each element in an array and returns an array of the manipulated element. So whereas the original array contained Yeezy's best albums, by invoking map on the array, and then invoking upcase on each element, we are returned a modified array where every element is upcase'd! Praise Yeezus.
