What is Big O Notation?

Big O notation is a way of expressing how long an algorithm takes to run. It is a measure of efficiency that allows comparisons of various solution. Instead of specifying the exact runtime of an algorithm, it is instead expressed as how quickly this runtime grows. This measurement of algorithmic efficiency is said.
